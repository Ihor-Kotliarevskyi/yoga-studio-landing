# Yoga Studio Landing

Лендінг для студії йоги, створений з використанням Vite та сучасних веб-технологій.

## Опис проекту

Цей проект є адаптивним лендінгом для йога-студії, який демонструє:

- Сучасний дизайн
- Анімації та інтерактивні елементи
- Адаптивність для мобільних та десктопних пристроїв

## Технології

- HTML5, CSS3, JavaScript (ES6+)
- Vite (збірка та дев-сервер)
- Система модулів ES
- Структура проекту з використанням `src` та `assets`

## Встановлення та запуск

1. Клонувати репозиторій:

```bash
git clone https://github.com/Ihor-Kotliarevskyi/yoga-studio-landing.git
```

2. Встановити залежності:

```bash
npm install
```

3. Запуск локального сервера:

```bash
npm run dev
```

4. Збірка проекту для продакшн:

```bash
npm run build
```

## Структура проекту

```
yoga-studio-landing/
├── assets/          # Статичні файли: зображення, шрифти
├── src/             # JS та CSS файли
├── .gitignore
├── index.html
├── package.json
├── vite.config.js
└── README.md
```

Проект можна розгорнути на GitHub Pages або будь-якому хостингу, який підтримує статичні сайти.
